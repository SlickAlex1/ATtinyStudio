# ATtinyStudio

ATtinyStudio is a lightweight IDE for programming ATtiny microcontrollers using Arduino IDE.

## Features
- Simple user interface.
- Supports multiple ATtiny microcontrollers.
- Easy installation.

## Setup Instructions
1. Download the latest version from the [releases page](https://github.com/SlickAlex1/ATtinyStudio/releases).
2. Follow the instructions in the `INSTALL.md` file to set up.

## Usage
- Connect your ATtiny microcontroller to the USBasp.
- Open ATtinyStudio and select your microcontroller from the menu.
- Write your code.
- Click upload to upload the code to your microcontroller.
